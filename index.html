Tentang Saya

Hai, Nama saya Muhamad Handy Fauzan, saya biasa dipanggil handy. Saat ini saya bersekolah di SMK SMAK Bogor, dan sekarang saya duduk di bangku kelas 11. Saya tinggal di Bojonggede, tepatnya di desa Kedung Waringin, saya tinggal bersama kedua orang tua saya dan adik saya. Saya adalah anak tunggal dari 2 bersaudara, dan saya memiliki 1 adik laki laki.

Selanjutnya hobi, Hobi saya adalah bermain Sepak Bola,saya mulai bermain sepak bola sejak berumur 4 tahun. Awalnya saya tau mengenai sepak bola dari om saya, emudian saya bermain game sepak bola dan seiring berjalannya waktu sepak bola seakan menjadi bagian dari hidup saya. Menurut saya Sepak Bola buaknlah hanya sekedar olahraga biasa, sepak bola memag terlihat mudah, tetapi dalam praktiknya tidak semua orang dapat bermain sepak bola, karena sepak bola tidak hanya mengandalkan skill individu tetapi juga memerlukan kerjasama tim yg baik dan kompak. Klub kesukaan saya adalah Real Madrid dan Persija Jakarta, Saya suka Real Madrid awalnya karena di Real Madrid ada pemain idola saya yaitu Raul Gonzales dan Iker Casillas, tapi lama kelamaan saya jadi sangan mencintai klub ini walaupun kedua idola saya sekarang sudah tidak bermain untuk Real Madrid. Sedangkan Saya menyukai Persija itu sejak saya kecil, mungkin karena ada darah betawi dari ayah saya yang membuat saya sangat cinta dengan Klub ini, Meskipun sudah 16 tahun tidak merasakan juara tetapi saya tetap mendukung klub kebanggaan masyarakat jakarta tersebut sampai kapanpun.
